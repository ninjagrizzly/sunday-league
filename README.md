## Setup

### Prerequisites

- Python 3.9+
- Poetry
- Telegram Bot Token

### Installation

1. Clone the repository:
```
git clone <repository-url>
cd football-tournament-bot
```

2. Install dependencies:
```
poetry install
```

3. Configure environment:
```
cp .env.example .env
```

4. Run the bot:
```
poetry run python main.py
```
